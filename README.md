# Loan-Risk-and-Approval-Analysis-using-MySQL-and-Power-BI
Data analytics project analyzing loan approvals and credit risk drivers through SQL processing and Power BI visualization.

Description:

- Designed and developed an end-to-end loan approval analysis dashboard using SQL and Microsoft Power BI to identify key factors influencing lending decisions.

Key Contributions:

- Cleaned and transformed raw loan data using SQL

- Engineeered analytical features including DTI ratio, income segmentation, and risk categories

- Performed exploratory data analysis to uncover approval patterns

- Built interactive Power BI dashboards for executive overview, risk analysis, and applicant segmentation

- Identified key approval drivers including credit risk, debt burden, and income stability

Tools & Skills: SQL, Power BI, Data Modeling, Risk Analysis, Data Visualization
